# ECHO

**ECHO** 是一个基于 NFC 的极简数字物件实验。  
它不是工具，也不是应用，而是一段被物件记住的时间。

当手机扫描 NFC 时，并不会立刻得到内容。  
只有在停留之后，才允许留下痕迹。

它更接近一本不能翻页的笔记，  
或一个不会回应、但会记住的东西。

---

## 功能（What it does）

- **NFC 扫描触发网页**
  - 芯片仅存储一个 URL
  - 无 App、无需安装

- **停留触发机制**
  - 页面需停留 ≥ 8 秒
  - 离开页面则自动取消

- **只追加，不修改**
  - 每次只能新增内容
  - 已有内容不可编辑、不可删除

- **有限记忆**
  - 最多记录 **7 条**
  - 达到上限后自动封存

- **封存后只读**
  - 不再接受新内容
  - 仅作为回看存在

- **无账号 / 无分享 / 无通知**
  - 不登录
  - 不同步
  - 不提醒

---

## 交互设计（How it feels）

- 无菜单、无返回、无多余提示  
- 页面内容随时间自然展开  
- 没有“成功”“失败”等反馈词  
- 所有输入都被视为**不可逆行为**

这个物件不催促用户行动，  
也不试图提高留存。

---

## 意义（Why it exists）

在数字世界中，大多数内容都是：

- 可复制的  
- 可修改的  
- 可删除的  
- 可被替代的  

**ECHO 刻意反其道而行：**

- 限制数量  
- 限制时机  
- 限制操作  
- 接受终点  

它尝试让一个数字行为拥有**重量**，  
让一次扫描不再只是触发，而是一种选择。

---

## 关于 NFC 安全性

- NFC 标签仅存储 URL  
- 不包含任何网页源代码或内容  
- 即使被复制，也无法篡改已存在的记录  

ECHO 不追求“绝对安全”，  
而是让复制本身**失去意义**。

## 适用场景

- 长期保留的实体物件  
- 私人纪念品  
- 实验性艺术作品  
- 关于时间、记忆与限制的探索  

---

## 一句话总结

> **ECHO 不是在记录内容，  
> 而是在记录你愿意为它停留的那一刻。**
